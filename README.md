## 📌 Project Overview

This project uses **Market Basket Analysis (MBA)** to identify relationships between products frequently purchased together. The analysis helps businesses improve cross-selling strategies, product recommendations, promotional campaigns, and store layouts.

The project applies the **Apriori algorithm** to discover frequent itemsets and generate association rules using retail transaction data.

---

## 🎯 Objectives

* Identify products frequently purchased together
* Generate association rules using the Apriori algorithm
* Analyze customer purchasing behavior
* Improve product recommendation systems
* Support data-driven business decisions

---

## 📊 Dataset Information

* **Dataset:** Market Basket Optimisation Dataset
* **Transactions:** 7,501
* **Products:** 120+

Each row represents a customer transaction containing multiple purchased products.

## 📈 Evaluation Metrics

The following metrics are used to evaluate association rules:

### Support

Measures how frequently an itemset appears in the dataset.

### Confidence

Measures the likelihood of purchasing one product when another product is purchased.

### Lift

Measures the strength of the association between products.
